## 🚀 Features

✔️ Stylish and responsive UI

## 🎯 How to Use
1️⃣ Click the buttons to enter numbers and operators.  
2️⃣ Press **"="** to calculate the result.  
3️⃣ Press **"C"** to clear the display.
